# Calculadora

Una aplicación web simple de calculadora construida con HTML, CSS y JavaScript.

## Características

- Operaciones básicas: suma, resta, multiplicación, división
- Interfaz intuitiva con botones numéricos y operadores
- Pantalla de visualización para ver la expresión y el resultado
- Botón de borrar (C) y borrar último dígito (⌫)

## Cómo usar

1. Abre `index.html` en tu navegador web.
2. Haz clic en los botones para ingresar números y operadores.
3. Presiona `=` para calcular el resultado.
4. Usa `C` para limpiar la pantalla o `⌫` para borrar el último carácter.

## Archivos

- `index.html`: Estructura de la página web
- `style.css`: Estilos para la interfaz
- `script.js`: Lógica de la calculadora

## Requisitos

- Un navegador web moderno que soporte JavaScript
